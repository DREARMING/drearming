##扫描界面的流程原理分析

背景：如果不懂Camera的工作方式，请查看官网 API Guide

  1. 我们先定义个surfaceView用来显示Camera拍到的图像
  2. 界面中那个ViewfinderView，是一个自定义控件，不断刷新，形成一个扫描动画，同时里面有一个hashset用来存储可能的结果点，在界面刷新时显示不同的颜色，给人一种好像扫描到东西的感觉。(备注：这些结果点是真的存在，是Camera扫到的可能结果，然后动态添加到这个自定义控件中，随着刷新显示出来)
